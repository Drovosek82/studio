# BMS Central — Інтелектуальна екосистема моніторингу

Цей додаток розроблений для моніторингу JBD BMS та ESP32 пристроїв з використанням штучного інтелекту Genkit.

## Як розгорнути та поділитися додатком

Для того, щоб інші користувачі могли протестувати додаток, виконайте ці кроки:

1. **Завантажте код у GitHub**:
   Створіть новий репозиторій у своєму GitHub та завантажте туди цей проект.

2. **Налаштуйте Firebase App Hosting**:
   - Перейдіть у [Firebase Console](https://console.firebase.google.com/).
   - Виберіть свій проект.
   - У меню зліва виберіть **Build > App Hosting**.
   - Натисніть **Get started** та підключіть свій GitHub-репозиторій.
   - Виберіть гілку (зазвичай `main`) та натисніть **Deploy**.

3. **Отримайте посилання**:
   Після завершення збірки Firebase надасть вам домен (наприклад, `your-project.web.app`), яким ви зможете поділитися з тестувальниками.

## Особливості спільного тестування

- **Приватність**: Кожен користувач бачить лише свої батареї (дані ізольовані за `userId`).
- **Спільний інтелект**: База знань моделей BMS (`modelInsights`) є глобальною. Якщо один тестувальник ідентифікує нову модель, вона з'явиться у списку знань для всіх інших.
- **Демо-режим**: Користувачі можуть натиснути "Демо" на головній сторінці, щоб побачити роботу інтерфейсу без реального заліза.
